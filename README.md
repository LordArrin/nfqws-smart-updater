# nfqws-smart-updater
Продвинутый скрипт обновления IP-листов для NFQWS (OpenWrt/(Keenetic/Entware tba)). Умная агрегация подсетей (CIDR merging), строгая валидация (sipcalc), кэширование, умное использование ресурсов устройства. Поддерживает cron и объединение локального файла с удалёнными источниками.
